Projects

YouTube Transcript Summarizer

Developed a Python-based web app that allows users to input a YouTube video link and get an instant transcript summary. The system extracts video transcripts using the youtube-transcript-api and applies NLP techniques like sentence tokenization, keyphrase extraction, and BERT-based summarization using Hugging Face Transformers. This tool helps users quickly understand long lectures, interviews, or tech talks. Built a simple UI using Streamlit and deployed it on the cloud.

Technologies: Python, NLP, HuggingFace Transformers, youtube-transcript-api, Streamlit

House Data Pipeline

Created a scalable ETL pipeline for housing price data extraction, transformation, and loading. Used BeautifulSoup and requests to scrape housing data from multiple real-estate websites, cleaned the data using Pandas, and stored it in an AWS Redshift warehouse. Implemented daily cron jobs for automation and added a validation module to handle inconsistent or duplicate entries. Added monitoring alerts using AWS SNS and integrated Tableau for dashboarding.

Technologies: Python, Pandas, AWS Redshift, SQL, Web Scraping, Tableau, AWS SNS
